Github+Hexo博客SEO优化

发表于 2018-10-18 | 阅读次数：

利用 github pages 搭建好自己的博客之后，最好自己买个域名解析过来，github 是屏蔽百度的爬虫的，也就是说，你辛苦的搭建起来的博客除了你自己并没有人看得到。所以我们需要进行一些优化，能够让百度搜索引擎搜得到，这样自己写的博客才有某种意义。

自定义域名

去域名服务商那边购买一个自己的域名，也不贵，也就几十块钱，遇上打折的时候可能就几块，十几块。

域名管理后台的解析配置

  • 腾讯云域名购买
  • 阿里旗下的万网域名购买

我是从腾讯那边注册的域名，下面以腾讯的域名管理为例，其他平台的都异曲同工

购买域名后进入管理后台，将 github pages 那边的目录添加解析

  • 域名解析列表☞域名☞操作[解析]
  • 在记录列表添加记录 ☞
    • 主机记录： www
    • 记录类型： CNAME
    • 记录值： github pages上原来分配的访问地址去掉https://。如：https://excample.github.io ,则记录值填写excample.github.io。其余的值默认，保存即可

github pages博客仓库的Settings的配置

github pages 这边的配置就比较简单了，在Settings配置页面
Github Pages☞Custom domain
然后在Custom domain 中填入 腾讯那边注册购买的域名,点击 Save

www.weilni.cn


上面配置好之后可以看到 github 仓库上会多了一个 CNAME的文件，内容显示：www.weilni.cn，即刚刚配置上去的域名↓↓


至此，自定义域名的解析以及配置完成，等上10来分钟就可以用自定义的域名进行自己博客的访问了。但是这只是自定义域名与 github pages 那边的绑定解析关系，Hexo 并未进行处理，因此接下来还要对 Hexo 进行简单的配置，保证之后写的文章提交时不会影响自定义域名的解析。

添加百度统计

百度统计管理后台配置

添加自己的网站

到百度统计注册百度统计-站长版。
注册后，进入百度统计的管理菜单依次找到

网站列表☞自有网站☞新建网站

将自己的博客站点添加进去，添加的时候直接添加自定的域名

获取百度统计ID

将自己的网站添加到百度统计之后，找到一下菜单，获取百度统计提供的ID

代码管理☞代码获取

在代码中找到hm.src="https://hm.baidu.com/hm.js?****"，这个 ****就是我们需要获取的百度统计的 ID

百度统计Hexo的配置

在 主题文件夹下打开 _config.yml 文件，找到Baidu Analytics，将百度统计的 ID 配置上去就 ok了

配置后用 Hexo命令进行生成页面与部署到 github 服务器上

然后去百度统计的管理后台进行验证代码，验证成功后就算是配置完成了。

以上，百度统计的相关配置已经完成，等上一段时间，再进入百度统计的个人中心即可查看的相关统计数据了。

安装 百度链接提交 hexo 插件

安装好插件后，在站点配置文件_config.yml配置站点的新字段：说明主动图送 链接提交的相关配置

配置后，还需要在站点配置文件_config.yml的 deploy: 字段下多添加一个以上配置的百度链接提交的配置类型

注意:

  • baidu_url_submit:配置一定要写这个名称，这是定式，否则【hexo g 会报错】
